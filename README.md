# 🎮 Video Game Sales Analysis Dashboard | Tableau

## Project Overview

This project explores global video game sales data using Tableau to identify sales trends across genres and years. The dashboard was designed to transform raw sales data into meaningful visual insights that help understand how the gaming industry has evolved over time.

The analysis focuses on two key business questions:

* Which video game genres generate the highest global sales?
* How have global video game sales changed over the years?

Using Tableau's visualization capabilities, the project presents a clear and interactive view of sales performance within the gaming market.
